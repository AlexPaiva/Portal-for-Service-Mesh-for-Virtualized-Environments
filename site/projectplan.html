Our description

This is our project description.

Currently, we are witnessing a globalization of web services. Services such as Instagram, Facebook, among many others, have to deal with users from all over the world. world, with a high level of performance.
A service that doesn't allow for near-instant updates, regardless of users' location, will not be popular. So, the service providers that provide the mechanisms and infrastructure needed to host, operate and offer these web services use distributed and flexible mechanisms.
One of these mechanisms is the use of virtualization infrastructures, which allow the installation of different components of a web application in a virtualized environment, which resides in a virtual machine or container.
This feature allows service providers to easily install and move components to different parts of the infrastructure, since these components do not need to be natively installed on a dedicated server. There are several offerings of virtualization engines, one of the most known today is called Kubernetes. Kubernetes is a system that allows automate and manage the installation of containerized software.
We'll be working with the last one, creating a webapp that allows anyone to manage Kubernetes clusters multiplaform and supply you with an easy to use interface.
